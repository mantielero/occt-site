This is an attempt to create bindings for the Nim programming language.

{{< /blocks/cover >}} {{% blocks/lead %}} occt.nim/a> aims to provide bindings for the Nim programming language and it adds a layer which simplifies the usage. It is currently in the early stages (just some basic examples working) but it is capable to export STEP files already. {{% /blocks/lead %}} {{< blocks/section >}}

This wrapping process has been done thanks to c2nim. Some experiments where done with cinterop. In the future, hcparse could be explored also.

{{< /blocks/section >}} {{< blocks/section >}}

The simplification layer is still very small.
